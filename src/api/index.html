Connect FareCraft to Claude Desktop

Use MCP to access flight search from Claude

What is MCP?

Model Context Protocol (MCP) allows Claude to connect to external tools and data sources. By adding FareCraft as an MCP server, Claude can search flights and calculate CPP directly from your conversations.

1 Find Your Claude Desktop Config File

macOS: ~/Library/Application Support/Claude/claude_desktop_config.json
Windows: %APPDATA%\Claude\claude_desktop_config.json
Linux: ~/.config/Claude/claude_desktop_config.json

2 Add FareCraft to Your Config

Add this configuration to your claude_desktop_config.json file: 

{
  "mcpServers": {
    "farecraft": {
      "command": "uv",
      "args": [
        "--directory",
        "/path/to/farecraft",
        "run",
        "python",
        "-m",
        "api.mcp_server"
      ]
    }
  }
}

‚ö†Ô∏è Important: Replace /path/to/farecraft with the absolute path to your FareCraft project directory (e.g., /home/user/WS).

üí° If you already have other MCP servers configured, just add the "farecraft" entry to your existing "mcpServers" object.

3 Restart Claude Desktop

Quit Claude Desktop completely and reopen it. The FareCraft MCP server will now be available in Claude.
